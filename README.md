# Exercises-for-introductory-artificial-intelligence-course-at-Mashhad-Azad-University
تمرین های درس هوش مصنوعی (پایتون: ۱۰ تمرین.


۱. سیستم مدیریت نمرات دانشجویان

بخش پایتون: تعریف کلاس یا دیکشنری برای ذخیره نام و نمرات.

بخش پانداز: تبدیل داده‌ها به DataFrame، محاسبه معدل کل کلاس و ذخیره خروجی در یک فایل CSV.

بخش نام‌پای: استفاده از آرایه‌ها برای اعمال ضریب‌های مختلف روی نمرات دروس خاص.

۲. تحلیل داده‌های هواشناسی (سنتتیک)

بخش نام‌پای: تولید یک آرایه شامل ۳۶۵ عدد تصادفی برای دمای روزهای سال (در بازه معین).

بخش پانداز: تبدیل آرایه به سری زمانی، پیدا کردن گرم‌ترین و سردترین روز سال و محاسبه میانگین دمای هر ماه.

۳. بهینه‌سازی سبد خرید فروشگاه

سناریو: لیستی از کالاها با قیمت و تعداد موجود دارید.

تمرین: با استفاده از NumPy، مجموع قیمت نهایی را با ضرب نقطه‌ای (Dot Product) آرایه‌های قیمت و تعداد محاسبه کنید. با Pandas، کالاهایی که موجودی آن‌ها زیر حد مجاز است را فیلتر و گزارش کنید.

۴. شبیه‌ساز پرتاب تاس و تحلیل آماری

بخش نام‌پای: شبیه‌سازی ۱۰۰۰ بار پرتاب دو تاس با استفاده از random.randint.

بخش پانداز: محاسبه فراوانی هر عدد (۲ تا ۱۲) و نمایش درصد احتمال رخ دادن هر مجموع به صورت جدول.

۵. پاکسازی داده‌های ناقص (Data Cleaning)

تمرین: یک فایل CSV حاوی داده‌های ناقص (NaN) به دانشجویان بدهید. آن‌ها باید با Pandas سطرهای خالی را حذف کنند، مقادیر عددی گم‌شده را با میانگین (محاسبه شده توسط NumPy) جایگزین کنند و نام ستون‌ها را استاندارد کنند.

۶. تحلیل تراکنش‌های بانکی

بخش پایتون: ورودی گرفتن تراکنش‌ها از کاربر.

بخش پانداز: دسته‌بندی تراکنش‌ها براساس نوع (خرید، واریز، برداشت) و محاسبه مجموع هر دسته. شناسایی تراکنش‌های مشکوک (بسیار بزرگتر از میانگین) با استفاده از توابع آماری نام‌پای.

۷. پردازش تصویر ساده (به عنوان ماتریس)

بخش نام‌پای: خواندن یک تصویر سیاه و سفید به صورت یک ماتریس دو بعدی. دانشجویان باید ماتریس را معکوس کنند (Invert) یا روشنایی آن را با جمع کردن یک عدد ثابت با کل آرایه تغییر دهند.

۸. سیستم رتبه‌بندی محصولات (Ranking System)

تمرین: داده‌های ۵ محصول شامل تعداد لایک و تعداد دیس‌لایک را در یک DataFrame ذخیره کنید.

فرمول: با استفاده از NumPy، یک امتیاز وزنی (مثلاً 
Score=Likes−(Dislikes×1.5)
) محاسبه کرده و محصولات را بر اساس آن مرتب کنید.

۹. تحلیل داده‌های بورس یا ارز دیجیتال

بخش پانداز: خواندن یک فایل تاریخچه قیمت. محاسبه "میانگین متحرک" ۵ روزه (Moving Average).

بخش نام‌پای: پیدا کردن روزهایی که قیمت بیش از ۳ درصد نوسان داشته است (استفاده از عملیات برداری برای سرعت بالا).

۱۰. نرمال‌سازی داده‌های علمی (Normalization)

تمرین: مجموعه‌ای از داده‌های عددی با مقیاس‌های متفاوت دارید.

بخش نام‌پای: پیاده‌سازی فرمول Min-Max Scaling فرمول را از اینترنت بررسی کنید. سپس کد ان را بنویسید.
